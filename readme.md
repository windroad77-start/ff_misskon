# ff_misskon (MissKon FlaskFarm Plugin)

MissKon.com 사이트의 고화질 화보 및 영상을 편리하게 감상하고, 단 한 번의 클릭으로 소장할 수 있는 FlaskFarm 전용 플러그인입니다. (`ff_4khd` 기반)

## 🌟 주요 기능

### 1. 다채로운 탐색 및 탭 구성
- **섹션별 독립 탭**: 최신(Latest) 및 인기 3일/7일/30일/60일 탭을 통해 원하는 콘텐츠에 즉시 접근 가능합니다.
- **스마트 검색**: 키워드를 통해 원하는 모델이나 테마의 화보를 빠르게 찾아낼 수 있습니다.
- **반응형 5열 그리드**: 데스크톱 환경에서는 한 줄에 5개씩, 모바일에서는 3개씩 자동으로 조절되는 최적화된 레이아웃을 제공합니다.

### 2. 고성능 통합 뷰어
- **자동 페이지 통합**: 여러 페이지로 분산된 워드프레스 게시물을 하나의 화면으로 통합하여 끊김 없이 감상할 수 있습니다.
- **고정형 다운로드 바**: 화보를 감상하는 도중 언제든 즉시 다운로드할 수 있도록 모달 하단에 버튼을 고정했습니다.
- **외부 다운로드 링크 수집**: 본문에 포함된 Terabox, Mega.nz 등 외부 클라우드 다운로드 링크를 자동으로 추출하여 하단에 표시해 줍니다.

### 3. 강력한 다운로드 엔진
- **실시간 진행률(%) 표시**: 다운로드 버튼에서 실시간으로 수집 상황(예: 25/50, 50%)과 상태(압축 중 등)를 확인할 수 있습니다.
- **자동 JPG 고화질 변환**: 원본 화질을 유지하며 **최고 화질(95%) JPG**로 자동 변환하여 저장합니다.
- **자동 ZIP 압축 및 정리**: 다운로드가 완료되면 폴더 전체를 ZIP 파일로 압축하고, 임시 이미지는 자동으로 삭제하여 저장 공간을 효율적으로 관리합니다.
- **중복 다운로드 방지**: 이미 받은 파일이 있을 경우 사용자에게 확인 메시지를 띄워 불필요한 재다운로드를 막아줍니다.

### 4. API 기반 고속 탐색
- **WP REST API 연동**: 최신(Latest) 및 검색(Search) 기능은 워드프레스 API를 직접 호출하여 매우 빠르고 정확한 결과를 제공합니다.
- **이미지 프록시**: 사이트 차단이나 핫링크 방지가 걸려 있어도 서버를 경유하여 안정적으로 이미지를 불러옵니다.

## ⚙️ 설정 방법

1.  **사이트 주소**: 현재 접속 가능한 MissKon 도메인을 입력합니다. (예: `https://misskon.com`)
2.  **이미지 프록시**: 이미지 로딩이 원활하지 않거나 엑스박스가 뜰 경우 활성화합니다.
3.  **다운로드 경로**: 화보 ZIP 파일이 저장될 서버의 로컬 경로를 설정합니다.

## 🛠 필수 라이브러리

본 플러그인은 아래의 라이브러리를 필요로 합니다:
- `cloudscraper`: 사이트 보안 우회 (HTML 파싱 시)
- `beautifulsoup4`: HTML 데이터 파싱
- `Pillow (PIL)`: 고품질 이미지 포맷 변환
